# C++ wrapper for LibPcap



## build tests and examples
- Build tests and examples 
```
mkdir build
cd build
cmake ..
make
```

- Run tests and examples:

```
from  build directory:
./runtests

```

To run an extra test case by openning a particular .pcap file:

```
./runtests [path_to_pcap_file]
```
